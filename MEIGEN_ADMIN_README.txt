v38 追記:
- 投稿案フォームの「送信」ボタンから /api/meigen-submit へ投稿案を送信します。
- Cloudflare Pages の KV binding は既存の SITE_STATS を利用できます。別管理にしたい場合は MEIGEN_SUBMISSIONS でも動作します。
- 管理ページのパスワード解除後、「投稿案リスト」で公開追加・非公開保存・削除ができます。
- KV binding が未設定の場合、投稿案は送信者のブラウザから管理ページへURL引き渡しされるだけで、全員共通の受信箱には保存されません。
